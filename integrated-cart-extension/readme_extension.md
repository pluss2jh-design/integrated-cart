# Integrated Cart Chrome Extension 사용 가이드

이 확장 프로그램은 쿠팡의 403 Forbidden 차단 및 B마트의 앱 전용 제약을 브라우저 상에서 우회하여 데이터를 가져올 수 있도록 돕습니다.

### 설정 방법
1. 크롬 브라우저에서 `chrome://extensions/`로 이동합니다.
2. 우측 상단의 **'개발자 모드'**를 활성화합니다.
3. **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭합니다.
4. 프로젝트 폴더 내의 `integrated-cart-extension` 폴더를 선택합니다.

### 주요 기능
- **쿠팡 우회**: 백엔드 서버 대신 사용자의 브라우저에서 직접 쿠팡을 검색하여 403 차단을 피합니다.
- **B마트 연동**: 모바일 웹 뷰를 통해 B마트의 실시간 데이터를 가져올 수 있는 브리지를 제공합니다.
- **통합 구매**: 검색된 최저가 재료들을 클릭 한 번으로 각 마트 장바구니에 담을 수 있게 지원합니다.
